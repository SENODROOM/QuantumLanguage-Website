Overview

Quantum is a tree-walk interpreted scripting language with a hand-written lexer, recursive descent parser, and 34 AST node types. It was built from scratch in C++17 by one person in 3 weeks. It works. Remarkably well, actually.

The language borrows syntax from Python, JavaScript, and C++ — and accepts all three simultaneously in the same file. This was a design choice. The parser has made peace with it.

hello.sa
# Python-style comment
// C++-style comment — both work

print("Hello from Quantum!")

# Three ways to declare a function — all valid:
fn greet(name):
    return "Hello, " + name

def greet2(name):
    return "Hi, " + name

function greet3(name) {
    return "Hey, " + name
}

print(greet("World"))
print(greet2("World"))
print(greet3("World"))
// three syntaxes, one language

Syntax Modes

Quantum accepts three distinct syntax styles. You can mix them freely. The parser handles it. The parser does not complain. The parser has simply accepted this as its life.

Python Style

Colons and indentation. def for functions. print(), and/or/not, True/False/None.

python-style.sa
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n - 1)

for i in range(1, 6):
    print(i, "! =", factorial(i))

JavaScript Style

Braces and semicolons. function keyword. let/const. console.log(). &&/||. Arrow functions with =>.

js-style.sa
const square = (x) => x * x;
let nums = [1, 2, 3, 4, 5];

for (n of nums) {
    console.log(n, "squared =", square(n));
}

C++ Style

Type annotations (int, float, string), pointer operators (&, *, ->), cout << output, new keyword. Type hints are decorative — everything is still dynamic underneath.

cpp-style.sa
int x = 42;        // stored as a double. always. this is fine.
string msg = "hello";

cout << x << endl;
cout << msg << endl;

int* ptr = &x;       // actual pointer. in a scripting language.
cout << *ptr << endl;
// 11 types, all in one std::variant

Control Flow

If / Elif / Else

# Python style (colon + indent):
if x > 10:
    print("big")
elif x > 5:
    print("medium")
else:
    print("small")

// C++/JS style (braces):

    console.log("big")

While Loop

while.sa
i = 0

For Loop

for.sa
# Python-style for-in:
for i in range(5):
    print(i)

# Iterate over array:
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# Dict iteration (key, value unpacking):
d = {"a": 1, "b": 2}
for k, v in d.items():
    print(k, "→", v)

# JS-style for-of:
for (x of fruits) {
    console.log(x)
}

Break & Continue

break-continue.sa
for i in range(10):
    if i == 3:
        continue   # skip 3
    if i == 7:
        break      # stop at 7
    print(i)       # prints 0 1 2 4 5 6
// three keywords, one concept

Functions

Functions are first-class values. They can be passed around, returned, and stored. Three keywords declare them: fn, def, and function. They are identical.

functions.sa
# Default arguments:
fn greet(name, greeting = "Hello"):
    return greeting + ", " + name + "!"

print(greet("Saad"))            # Hello, Saad!
print(greet("Saad", "Hey"))  # Hey, Saad!

# Pass by reference (C++ style):
fn increment(int& val):
    val += 1

x = 10
increment(x)
print(x)  # 11 — x was modified in place

# Functions as values:
fn apply(f, val):
    return f(val)

fn double(x): return x * 2

print(apply(double, 5))  # 10
// anonymous functions and closures

Lambdas & Closures

lambdas.sa
# Arrow function (JS style):
square = (x) => x * x
print(square(5))  # 25

# Multi-param arrow:
add = (a, b) => a + b

# Lambda keyword (Python style):
double = lambda x: x * 2
print(double(7))  # 14

# Closure — captures outer scope:
fn make_counter():
    count = 0
    fn inc():
        count += 1
        return count
    return inc

counter = make_counter()
print(counter())  # 1
print(counter())  # 2
print(counter())  # 3
// full object-oriented programming

Classes & OOP

Quantum has a complete OOP system: inheritance with extends, super() calls, static methods, and __str__. Both self and this refer to the instance.

oop.sa
class Animal:
    fn init(name, sound):
        self.name = name
        self.sound = sound

    fn speak():
        return self.name + " says " + self.sound

    fn __str__():
        return "Animal(" + self.name + ")"

class Dog extends Animal:
    fn init(name):
        super(name, "Woof")   # call parent init
        self.tricks = []

    fn learn(trick):
        self.tricks.append(trick)

    fn show_tricks():
        for t in self.tricks:
            print(self.name, "knows:", t)

dog = Dog("Rex")
print(dog.speak())   # Rex says Woof
dog.learn("sit")
dog.learn("fetch")
dog.show_tricks()

# Static methods:
class MathUtils:
    static fn square(x):
        return x * x

print(MathUtils.square(7))  # 49

Pointers

Quantum has a real pointer system. &var returns a QuantumPointer holding a live shared reference to the variable's storage cell. *ptr dereferences it. ptr->member accesses a field through a pointer. Null checks are enforced. This is a scripting language. We are aware.

pointers.sa
x = 42
ptr = &x              # AddressOfExpr → QuantumPointer

print(ptr)            # 0x7ffe8a3c1040 (deterministic fake address)
print(*ptr)           # 42 — dereference

*ptr = 99             # write through pointer
print(x)              # 99 — x was updated

# Class instance through pointer:
class Point:
    fn init(x, y):
        self.x = x; self.y = y

p = Point(3, 4)
pp = &p
print(pp->x)          # 3 — arrow operator

# Null pointer:
null_ptr = nil
# *null_ptr → RuntimeError: Null pointer dereference

Error Handling

errors.sa
# Python style:
try:
    x = 10 / 0
except RuntimeError as e:
    print("caught:", e)
finally:
    print("always runs")

# Raising errors:
fn check_age(age):
    if age < 0:
        raise "Age cannot be negative"
    return age

# JS-style try/catch:
try {
    check_age(-5)
} catch (e) {
    console.log("Error:", e)
}

Security™

Quantum's flagship cryptographic suite. rot13 is the only shipped cipher. The rest are reserved keywords waiting for v2. Caesar used rot13. Caesar was subsequently stabbed by his friends.
